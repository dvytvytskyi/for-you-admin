# For You Real Estate App

Мобільний додаток та веб-платформа для нерухомості.

## Структура проекту

```
for-you-app/
├── mobile/              # React Native мобільний додаток
├── backend/             # NestJS backend API
├── admin-backup/        # Заморожена адмін панель (не використовується)
└── docker-compose.yml   # Docker конфігурація
```

## Технології

- **Mobile:** React Native (Expo)
- **Backend:** NestJS, PostgreSQL, Redis
- **Admin Panel:** Next.js (заморожено в admin-backup/)

## Розробка

Дивіться документацію в окремих папках:
- `mobile/README.md` - для мобільного додатку
- `backend/README.md` - для backend API
